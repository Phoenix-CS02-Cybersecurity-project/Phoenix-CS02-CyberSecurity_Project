Macchanger
Kali linux mac changer python script

Usage :

Commands

python macchangerpy.py --i "interface_name" -m "new_mac address"

or

python macchangerpy.py --interface "interface_name" --new_mac "new_mac address"

example :

python macchangerpy.py --interface eth0 -m 00:11:22:33:44:55
